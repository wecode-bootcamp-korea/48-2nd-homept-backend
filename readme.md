## ****🏋️‍♀️**** 서비스 소개

### 기획 의도

바쁜 현대인들을 위한 피트니스와 운동 서비스를 기획하여 시간과 공간의 제약 없이 건강한 라이프스타일을 추구하는 모든 이들에게 편의성과 효과성을 제공

### 핵심 기능

로그인

- 이메일 중복여부 확인
- 토큰 발급
- 유저 타입 구분

내 정보 및 마이페이지 수정

커뮤니티・상담 게시글

- 데이터 정렬
- 날짜 포맷팅
- 사진 첨부 기능

결제

- 외부 API
- 포인트 결제
